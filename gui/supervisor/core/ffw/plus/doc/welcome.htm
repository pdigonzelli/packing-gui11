Welcome to the Webspeed Pluspack

QuickStart:
You should now have two more options on the Webtools menu. The databrowser give you browse functionality for currently connected databases. The appman "application manager" you need to configure before you can start using, please do so by clicking the two config hyperlinks.

Improvements:

  • Does not require compilation.
  • Can be installed anywhere on the propath. (just copy/unzip the plus/ directory + subfolders)
  • Has new images and static help files. (Images only work from local browser)
  • Bugfixes to the startup enhancer / debugging system.

Hints:

  • Please compile the pluspack for improved performance.
