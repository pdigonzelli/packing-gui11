Application Manager: System Config

System Config makes it easy to set up the Broker and Database part of the Application Manager.

Simply select the services and databases you want to add to the main menu and Click the "OK" button to accept and return to main menu.

 

Searches Registry on NT for ProControl settings.

  1. Databases configured in ProControl

Searches "conmgr.properties" for database settings.

  1. Databases configured in Progress Explorer

Searches "ubroker.properties" for services settings.

  1. Name Servers
  2. Webspeed Brokers
  3. App Server Brokers
  4. Oraserver Brokers
  5. ODBC Server Brokers

Add unlisted servers.

This let you add servers that don't appear in the list. This is often applicable for databases that is not served by the Adminservice.
